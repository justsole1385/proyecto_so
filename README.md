# Proyecto de UCOM 418 - Sistemas Operativos (Paralelo 81)

# Descripción y Tema
Proyecto Final de la materia UCOM 418 - Sistemas Operativos (Paralelo 81). El proyecto consiste en una simulación de una plataforma donde usuarios reservan asientos en una sala de cine, garantizando la correcta asignación y control de disponibilidad.
